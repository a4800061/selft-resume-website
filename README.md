## selft-resume-website
selft-resume-website，用react开发的简单的个人简历网站。适合react新手入门练习。
![site](./src/assets/images/site.png)

## 本地开发  
```js
npm start
```
## 打包构建
```
npm run build
```
打包后根目录下会出现一个`build`文件夹，这里面的代码就是你最终要部署的代码了。

把build文件夹上传到服务器部署即可。

或者使用`github pages`、`netlify`、`gitlab pages`免服务器免费部署。
